# Flight Discovery Frontend

This is a flight discovery frontend built with Next.js 14, TailwindCSS.

## Setup

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Create a `.env.local` file based on `.env.local.example` and set the `NEXT_PUBLIC_API_URL`.
4.  Start the development server: `npm run dev`

## API

The frontend fetches data from `http://localhost:8000/api/search`. Make sure the API is running locally.

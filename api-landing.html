Get Started in 60 Seconds

Step 1: Get your free API key

curl -X POST "https://flight-api.railway.app/api/keys/create" \ -H "Content-Type: application/json" \ -d '{"email": "you@example.com", "tier": "free"}'

Step 2: Search for flights

curl "https://flight-api.railway.app/api/search?origin=YUL&destination=LHR" \ -H "X-API-Key: YOUR_API_KEY"
